интернет магазин маркетплейс

функционал:
авторизация/регистрация
просмотр товаров и добавление товаров

апи:
POST запрос авторизации
url: localhost/auth/login
params:
login,pass

POST запрос регистрации
url: localhost/auth/registration
params:
login,pass,username

GET запрос товаров
url: localhost/item/list
params:
limit, offset

POST запрос добавления товаров
url: localhost/item/add
params:
title, description, price

Примечание
авторизованный юзер может добавить товар и просматривать списки товаров
не авторизованный может только просматривать список товаров

сущности
user:
    id, login, password, name, token
item:
    id, title, description, user_id, price
